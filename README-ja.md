Claude Code などのインタラクティブな AI CLI を包む、**完全透過の PTY プロキシ**。AI が考え終わった瞬間に叫んでくれる。

---

## これは何か

AI を起動する。コードを書き始める。シリコンの静寂の中で、歯車が回り続ける。

あなたはその場を離れる。ヘッドフォンをつけ、よそ見をして、機械に対して耳が聞こえないふりをする。タブの切り替えなし。不安なポーリングなし。「もう終わった？」もなし。

AI がプロンプトに戻った瞬間——次の命令を待つその瞬間——`crai` が沈黙を打ち破る。一つのチャイム。ガラスが割れる音。

**それが合図だ。戻っておいで。**

```sh

```

ソースからビルドする場合：

```sh

cd crai
go build -o crai .
sudo mv crai /usr/local/bin/
```

## 使い方

```sh
# claude を直接ラップする
crai claude

# 引数はそのまま透過される
crai claude --dangerously-skip-permissions
```

すべてそのまま透過される。色、スピナー、キーバインド——すべて無傷。`crai` は必要な時まで見えない。

---

## エイリアス設定

シェルの設定ファイル（`~/.zshrc` または `~/.bashrc`）に追加：

```sh
alias claude="crai claude "
```

> **末尾のスペースについて**
> bash と zsh では、エイリアスの値の末尾にスペースがあると、次の単語もエイリアスとして展開される。これにより、`claude` の後に渡す引数もエイリアス展開の対象になり、エイリアスチェーンが完全に機能する。

あとはいつも通り `claude` を使うだけ。`crai` は静かに見張っている。

## 仕組み

```
 ┌──────────┐   raw stdin   ┌─────────────┐   PTY   ┌───────────┐
 │  あなた  │ ────────────► │    crai     │ ──────► │  claude   │
 │  (人間)  │ ◄──────────── │  (watcher)  │ ◄────── │  (AI CLI) │
 └──────────┘   raw stdout  └──────┬──────┘         └───────────┘
                                   │
                   AI 出力から 1500ms 以上の沈黙
                                   │
                                   ▼
                    🔔 afplay Glass.aiff
                    🪟 通知センターバナー
                    🔕 ターミナルベル (\a)
```

1. コマンドを**疑似端末（PTY）**の中で起動する
2. raw な stdin/stdout を変換なしでブリッジする
3. 出力ストリームの**沈黙**を監視——1500ms 以上新しい出力がなければ AI が完了したと判断
4. 完了時：サウンド、通知センターバナー、ターミナルベルの3つを同時に発火
5. 沈黙に戻る。待つ。見張る。

### スマートフィルタリング

- **1対1プロンプト対応** — Enter を押すたびに通知が1つだけ予約される。対応するプロンプトのない AI 出力（起動時のバナー、自発的な出力）は無視される
- **エコー除去** — キー入力から 100ms 以内に届いた出力は PTY のエコーとして扱い、AI 出力とみなさない
- **短時間応答の抑制** — AI が 5 秒以内に応答した場合は通知しない（まだ見ているはず）
- **入力中の抑制** — 次のメッセージを作成中は通知しない

## 名前の由来 / 世界観

`crai` という名前には、同時に3つの意味が込められている。

### 1. **C**atcher in the **rAI**
J.D. サリンジャーの『ライ麦畑でつかまえて』へのオマージュ。『攻殻機動隊 S.A.C.』に登場する「笑い男」こと荒巻が、自らの思想の拠り所として携えていた小説だ。彼はその冒頭の一節を企業ロゴに埋め込んだ——気づいた者だけに見えるように。`crai` も同じだ。必要な時まで見えない。

### 2. 笑い男 vs. 泣き男
荒巻は*笑い男*だった——沈黙し、仮面をつけ、追跡不可能な存在。このツールはその影：**泣き男**。彼が沈黙を抱きしめたところで、`crai` は沈黙を破る。ついに語らなかった亡霊の、憂鬱な対極。

### 3. 叫ぶ（Crying Out）
ツールの機能を端的に言えば：**叫ぶ**こと。AI が考え終わってプロンプトに戻った瞬間、`crai` は「ねえ、終わったよ」と声を上げる。
